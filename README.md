# Potret-Ketimpangan-Sosial-Ekonomi-Jakarta-2019-2023

Ketimpangan sosial ekonomi merupakan tantangan utama dalam proses pembangunan perkotaan, terutama di wilayah metropolitan seperti Jakarta. Meskipun secara agregat Jakarta menunjukkan kemajuan ekonomi yang signifikan, pemerataan hasil pembangunan antarwilayah masih menjadi isu yang belum terselesaikan. Jakarta terdiri dari lima kota administratif dan satu kabupaten administratif (Kepulauan Seribu), yang menunjukkan karakteristik sosial ekonomi yang sangat beragam. Dashboard Analisis ini dibuat untuk mengeksplorasi data sosial ekonomi antarwilayah di Jakarta dalam periode 2019–2023
